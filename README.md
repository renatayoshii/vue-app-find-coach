## About This Project

This project was developed as part of the Vue.js course taught by Maximilian Schwarzmüller.

The main goal was to practice core Vue.js concepts, including:

* Component-based architecture
* State management with Vuex
* Routing with Vue Router
* Authentication
* HTTP requests and Firebase integration
* Form handling and validation

While following the course structure, this repository serves as a personal learning project and a demonstration of my Vue.js skills.

## Learning Objectives

* Build scalable Vue.js applications
* Manage application state efficiently
* Work with external APIs and databases
* Implement authentication and route protection
* Apply Vue.js best practices
